Setup

  • Go to Setup > Email > Email Account
  • Create a new Email Account
  • Select Service as Mandrill
  • Set your Mandrill Username and Mandrill API Key as the password
  • On saving the Email Account, Mandrill Integration app will attempt to create a Webhook in your Mandrill account

Note: Your ERPNext site should be accessible to outside servers i.e. Mandrill\'s servers should be able to send requests to your ERPNext site
